# 소수구하기
> 2부터 n-1 까지의 어떤 수로도 나누어 떨어지지 않는 수
>
>`n % (2 ~ n-1) != 0`
>
> 2부터 n-1 까지의 어떤 소수로도 나누어 떨어지지 않는 수
> 
>> 2또는 3으로 나누어 떨어지지 않으면 2와 3의 배수로도 나누어 떨어지지 않는다.
>> `n % primeNumber != 0`
> 
> n의 제곱근 이하의 어떤 소수로도 나누어 떨어지지 않는다.
>
