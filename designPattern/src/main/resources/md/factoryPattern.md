# 팩토리

Factory 디자인 패턴은 생성 디자인 패턴으로, 객체를 만드는 가장 좋은 방법 중 하나입니다. 
해당 패턴은 Factory 메서드를 사용하여 객체의 정확한 클래스를 정의하지 않고 객체 생성 문제를 처리합니다.
